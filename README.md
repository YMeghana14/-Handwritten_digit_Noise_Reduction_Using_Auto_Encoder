# -Handwritten_digit_Noise_Reduction_Using_Auto_Encoder

          This project's aim is to recognise the handwritten digits using Deep learning techniques(Encoder and Decoder)

        ***** ENCODING AND DECODING THE IMAGES *****
        ***** TRAINING THE DATA *****
